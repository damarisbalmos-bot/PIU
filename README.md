# Aplicație Inventar Baterii

## Descriere

Această aplicație de consolă este destinată gestionării unui inventar de baterii. Permite utilizatorilor să adauge baterii în inventar, să vizualizeze bateriile existente și să le șteargă. Este o aplicație simplă și eficientă pentru urmărirea bateriilor disponibile într-un depozit sau într-o locație de stocare.

## Funcționalități

- **Adăugare baterie:** Permite utilizatorilor să adauge o nouă baterie în inventar, specificând detalii precum numele bateriei, tipul acesteia, data de expirare și cantitatea disponibilă.
- **Vizualizare inventar:** Permite utilizatorilor să vizualizeze toate bateriile aflate în stoc, cu detalii complete despre fiecare.
- **Ștergere baterie:** Permite utilizatorilor să șteargă o baterie din inventar, căutând-o după numele acesteia.
